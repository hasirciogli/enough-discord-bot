## Run Locally

Clone the project

```bash
  git clone https://github.com/hasirciogli/enough-discord-bot
```

Go to the project directory

```bash
  cd enough-discord-bot
```

Install dependencies

```bash
  npm install
```

Start the bot

```bash
  node server
```
